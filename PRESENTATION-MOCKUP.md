## Data for SQL/Python Practice (Sunday)

Good datasets to query:
1. **CFPB Consumer Complaints** — filter by product (credit cards, payday, BNPL), group by state, demographics, issue type
2. **FRED API** — pull delinquency rates over time, consumer credit totals
3. **Census/ACS** — income by ZIP code, education levels, correlate with credit data

Example queries:
```sql
-- Top complaint types for credit cards
SELECT issue, COUNT(*) as complaints
FROM consumer_complaints
WHERE product = 'Credit card'
GROUP BY issue
ORDER BY complaints DESC
LIMIT 10;

-- Complaints by state
SELECT state, COUNT(*) as total
FROM consumer_complaints
WHERE product LIKE '%payday%'
GROUP BY state
ORDER BY total DESC;
```

```python
# Python: Plot debt over time
import matplotlib.pyplot as plt

years = [2003, 2006, 2009, 2012, 2015, 2018, 2020, 2021, 2023, 2025]
debt_trillions = [0.6, 0.75, 0.85, 0.82, 0.9, 1.0, 0.82, 0.87, 1.13, 1.28]

plt.figure(figsize=(10, 6))
plt.plot(years, debt_trillions, marker='o', linewidth=2, color='#e74c3c')
plt.fill_between(years, debt_trillions, alpha=0.1, color='#e74c3c')
plt.title('US Credit Card Debt (2003-2025)', fontsize=16)
plt.ylabel('Trillions ($)', fontsize=12)
plt.xlabel('Year', fontsize=12)
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('debt_over_time.png', dpi=150)
plt.show()
```
